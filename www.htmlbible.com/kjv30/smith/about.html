To Install the Smith's Bible Dictionary

After downloading, create a folder inside the HTML Bible folder on your c: drive (that is, make this folder a sub-folder of the HTML Bible.) Name this folder "smith" - all lower case. It is very important that you name the folder correctly.

Unzip the two download files using WinZip or equivalent. Put the files in the Smith folder.

If you install the Smith folder under the Parallel New Testament HTML Bible, or in the Greek New Testament HTML Bible, the Old Testament references will not work.

To Use the Smith Bible Dictionary

Open the index.htm file in the Smith folder with your browser.
