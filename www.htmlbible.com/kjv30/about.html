How to Install and Un-install This Software

To Install

The King James Reference Suite in Microsoft HTML Help format is contained within the kjv.chm, easton.chm, nave.chm, torrey.chm, henry.chm, and kjvimages.chm files.

To Install:

1. Place the .chm files in a folder on your PC:

2. Open the kjv.chm file with your browser.
Installation is now complete.
You can make a shortcut to the kjv.chm file on your desktop, or folder, as needed.

To Un-Install

1. Delete the kjv.chm and other .chm files. There have been no updates made to your registry by this software.

2. Delete any shortcuts you have made.

3. The software has been un-installed.
